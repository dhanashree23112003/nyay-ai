## рҹҡЁ The Problem

India has **1.4 billion people**. Most of them speak neither English nor legalese.

Filing a First Information Report means navigating a Byzantine system of IPC sections, police procedures, and paperwork вҖ” in a language you may not fully understand. Officers manually cross-reference hundreds of sections. Errors happen constantly. Cases get filed under the wrong sections. Justice delayed is justice denied.

**Nyay AI fixes this.** Speak naturally. In Hindi. In English. In Hinglish. The AI figures out the rest.

## рҹҗӣ The Bug That Was Caught & Fixed (Engineering Transparency)

Multi-turn conversations were **silently breaking**. The model kept repeating questions it had already asked. Took a while to find why.

**Root cause:** Python's `str()` on dicts mangles unicode characters.

```python
# вқҢ THE BAD VERSION (mangled unicode, broken context)
history.append({
    "role": "assistant",
    "content": str(result)  # \u0930\u093e\u092e instead of аӨ°аӨҫаӨ®
})
# LLM sees garbled nonsense вҶ’ repeats questions вҶ’ user confused вҶ’ FIR broken

# вң… THE FIX (unicode preserved, context intact)
history.append({
    "role": "assistant",
    "content": json.dumps(result, ensure_ascii=False)  # аӨ°аӨҫаӨ® вң“
})
# Multi-turn context works вҶ’ no repeat questions вҶ’ FIR complete
```

Found it. Fixed it. **Disclosed it.** That's how engineering should work.
